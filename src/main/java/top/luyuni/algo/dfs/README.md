# DFS 目录 - 深度优先搜索专题 (Depth First Search)

这个目录帮你彻底理解DFS，从「递归的本质」到「DFS的实战应用」。

## 为什么学DFS？

DFS是解决「遍历/搜索所有可能」的神器：

- 二叉树的遍历（前序、中序、后序）
- 图的遍历（连通分量、路径搜索）
- 排列组合问题（全排列、子集）
- 回溯算法（N皇后、数独）
- 岛屿问题（网格连通块）

## DFS的核心概念

### 什么是DFS？

```
DFS（深度优先搜索）：
- 沿着一条路径一直走到底，走不通再回头
- 用「栈」实现（递归的本质就是栈）

形象比喻：
- 走迷宫：一条路走到黑，遇到死胡同就退回上一个岔路口
- 剥洋葱：一层一层往里剥，剥到最里面再出来

DFS vs BFS：
- DFS：深度优先，一条路走到底（用栈/递归）
- BFS：广度优先，一层一层扩展（用队列）
```

### DFS的两种实现方式

```java
// 方式1：递归（最常用）
void dfs(节点) {
    if (终止条件) return;
    
    访问当前节点;
    标记已访问;
    
    for (每个邻接节点) {
        if (未访问) {
            dfs(邻接节点);
        }
    }
}

// 方式2：显式栈（非递归）
void dfs(起始节点) {
    Stack<节点> stack = new Stack<>();
    stack.push(起始节点);
    
    while (!stack.isEmpty()) {
        节点 = stack.pop();
        
        if (未访问) {
            访问节点;
            标记已访问;
            
            // 子节点入栈（注意顺序，反着入）
            for (每个邻接节点，倒序) {
                if (未访问) {
                    stack.push(邻接节点);
                }
            }
        }
    }
}
```

### DFS的框架模板

```java
// 通用DFS框架
class DFS {
    // 访问标记（防止重复访问）
    boolean[] visited;
    
    // 主入口
    void dfs(参数) {
        // 初始化
        visited = new boolean[大小];
        
        // 从起点开始DFS
        dfsHelper(起点);
    }
    
    // 递归函数
    void dfsHelper(当前节点) {
        // 1. 终止条件
        if (终止条件) return;
        
        // 2. 访问当前节点
        visited[当前节点] = true;
        process(当前节点);
        
        // 3. 遍历邻接节点
        for (邻接节点 : 获取邻接节点(当前节点)) {
            if (!visited[邻接节点]) {
                dfsHelper(邻接节点);
            }
        }
        
        // 4. 回溯（如果需要）
        // visited[当前节点] = false;  // 视情况而定
    }
}
```

## DFS的经典应用场景

### 场景1：二叉树的DFS遍历

```
前序遍历：根 → 左 → 右
中序遍历：左 → 根 → 右
后序遍历：左 → 右 → 根

示例树：
    1
   / \
  2   3
 / \
4   5

前序：1, 2, 4, 5, 3
中序：4, 2, 5, 1, 3
后序：4, 5, 2, 3, 1
```

### 场景2：图的连通分量

```
问题：找无向图中的连通分量个数

思路：
1. 遍历所有节点
2. 如果节点未访问，从该节点开始DFS
3. DFS访问到的所有节点属于同一个连通分量
4. 连通分量数++

示例：
0 -- 1    3 -- 4
|         |
2         5

连通分量1：0, 1, 2
连通分量2：3, 4, 5
答案：2
```

### 场景3：岛屿问题（网格DFS）

```
问题：给定二维网格，1表示陆地，0表示水，求岛屿数量

示例：
11110
11010
11000
00000

岛屿：1的连通块（上下左右相邻）

DFS思路：
1. 遍历每个格子
2. 遇到1，岛屿数++，然后开始DFS
3. DFS把连通的1都变成0（标记访问）
4. 继续遍历

过程：
(0,0)=1 → 岛屿1，DFS把连通的1都变成0
(0,4)=0 → 跳过
... → 没有其他1了

答案：1
```

### 场景4：全排列（回溯）

```
问题：给定数组[1,2,3]，求所有排列

DFS + 回溯思路：
1. 维护一个当前路径
2. 维护一个used数组，标记哪些元素已用
3. 路径长度=n时，找到一个排列
4. 回溯：撤销选择，尝试其他路径

过程：
[]
├─ [1]
│  ├─ [1,2]
│  │  └─ [1,2,3]  ✓
│  └─ [1,3]
│     └─ [1,3,2]  ✓
├─ [2]
│  ├─ [2,1]
│  │  └─ [2,1,3]  ✓
│  └─ [2,3]
│     └─ [2,3,1]  ✓
└─ [3]
   ├─ [3,1]
   │  └─ [3,1,2]  ✓
   └─ [3,2]
      └─ [3,2,1]  ✓

共6种排列
```

### 场景5：子集问题

```
问题：给定数组[1,2,3]，求所有子集

DFS思路：
每个元素有两种选择：选 或 不选

过程：
[]
├─ 不选1
│  ├─ 不选2
│  │  ├─ 不选3 → []      ✓
│  │  └─ 选3   → [3]     ✓
│  └─ 选2
│     ├─ 不选3 → [2]     ✓
│     └─ 选3   → [2,3]   ✓
└─ 选1
   ├─ 不选2
   │  ├─ 不选3 → [1]     ✓
   │  └─ 选3   → [1,3]   ✓
   └─ 选2
      ├─ 不选3 → [1,2]   ✓
      └─ 选3   → [1,2,3] ✓

共8个子集（2^3）
```

## DFS的关键技巧

### 技巧1：访问标记

```java
// 方式1：boolean数组（适用于节点编号0~n-1）
boolean[] visited = new boolean[n];

// 方式2：Set（适用于任意类型）
Set<节点类型> visited = new HashSet<>();

// 方式3：修改原数组（适用于网格问题）
grid[i][j] = '0';  // 把访问过的陆地变成水
```

### 技巧2：方向数组（网格问题）

```java
// 上下左右四个方向
int[][] dirs = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};

// 遍历四个方向
for (int[] dir : dirs) {
    int ni = i + dir[0];
    int nj = j + dir[1];
    // 检查边界和访问状态
}
```

### 技巧3：回溯（撤销选择）

```java
void dfs(路径, 选择列表) {
    if (满足结束条件) {
        记录结果;
        return;
    }
    
    for (选择 : 选择列表) {
        做选择;
        dfs(路径, 选择列表);
        撤销选择;  // 回溯的关键！
    }
}
```

## DFS学习建议

1. **先掌握递归**：理解递归的执行过程（调用栈）
2. **再写DFS框架**：记住"访问-递归-回溯"的套路
3. **多做岛屿题**：网格DFS是面试高频
4. **理解回溯**：撤销选择是关键
