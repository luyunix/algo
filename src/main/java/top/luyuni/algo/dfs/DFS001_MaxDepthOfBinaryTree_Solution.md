# DFS001 - 二叉树的最大深度 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：求二叉树的最大深度

示例树：
    3
   / \
  9  20
    /  \
   15   7

深度：根节点3到最远叶子15或7的路径长度
3 → 20 → 15，共3个节点，深度为3
```

### 第二步：发现递归关系

```
关键洞察：
- 树的深度 = 子树的深度 + 1
- 最大深度 = max(左子树深度, 右子树深度) + 1

示例：
节点3的深度 = max(节点9的深度, 节点20的深度) + 1

节点9的深度：
- 左子树空，深度0
- 右子树空，深度0
- 节点9深度 = max(0, 0) + 1 = 1

节点20的深度：
- 左子树15深度 = 1
- 右子树7深度 = 1
- 节点20深度 = max(1, 1) + 1 = 2

节点3深度 = max(1, 2) + 1 = 3 ✓
```

### 第三步：确定终止条件

```
终止条件：当前节点为空
- 空节点的深度 = 0
```

---

## 解法：递归DFS

```java
public int maxDepth(TreeNode root) {
    // 终止条件：空节点深度为0
    if (root == null) return 0;
    
    // 递归求左右子树深度
    int leftDepth = maxDepth(root.left);
    int rightDepth = maxDepth(root.right);
    
    // 当前节点深度 = max(左, 右) + 1
    return Math.max(leftDepth, rightDepth) + 1;
}
```

## 边界 Case

**Case 1: 空树**
```
root = null
直接返回0 ✓
```

**Case 2: 只有根节点**
```
    1

左右都是null，返回max(0,0)+1 = 1 ✓
```

**Case 3: 只有左子树**
```
    1
   /
  2
 /
3

节点3: 返回1
节点2: max(1,0)+1 = 2
节点1: max(2,0)+1 = 3 ✓
```

**Case 4: 只有右子树**
```
1
 \
  2
   \
    3

节点3: 返回1
节点2: max(0,1)+1 = 2
节点1: max(0,2)+1 = 3 ✓
```

**Case 5: 满二叉树**
```
    1
   / \
  2   3
 / \ / \
4  5 6  7

节点4,5,6,7: 都返回1
节点2: max(1,1)+1 = 2
节点3: max(1,1)+1 = 2
节点1: max(2,2)+1 = 3 ✓
```
