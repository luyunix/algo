# DFS003 - 岛屿数量 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：给定二维网格，1表示陆地，0表示水，求岛屿数量

示例1：
11110
11010
11000
00000

岛屿：相邻（上下左右）的1组成的连通块
答案：1（所有1都连通）

示例2：
11000
11000
00100
00011

岛屿1：左上角的4个1
岛屿2：中间的1个1
岛屿3：右下角的2个1
答案：3
```

### 第二步：发现规律

```
关键洞察：
- 遍历每个格子
- 遇到1，说明发现新岛屿
- 从这个1开始DFS，把所有连通的1都访问一遍
- 访问过的1要标记（变成0），避免重复计数

为什么用DFS？
- 找到一个陆地后，要找到所有连通的陆地
- 这是典型的连通块问题，DFS最自然
```

### 第三步：设计算法

```
算法步骤：
1. 遍历网格的每个格子
2. 如果grid[i][j] == '1'：
   a. 岛屿数++
   b. 从(i,j)开始DFS，把连通的'1'都变成'0'
3. 返回岛屿数

DFS函数：
- 标记当前格子为'0'（已访问）
- 向四个方向扩散（上下左右）
- 对每个方向，如果在边界内且是'1'，继续DFS
```

---

## 解法：网格DFS

```java
public int numIslands(char[][] grid) {
    if (grid == null || grid.length == 0) return 0;
    
    int count = 0;
    int m = grid.length, n = grid[0].length;
    
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (grid[i][j] == '1') {
                count++;
                dfs(grid, i, j);
            }
        }
    }
    
    return count;
}

private void dfs(char[][] grid, int i, int j) {
    int m = grid.length, n = grid[0].length;
    
    // 越界或不是陆地，返回
    if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] != '1') {
        return;
    }
    
    // 标记为已访问（变成水）
    grid[i][j] = '0';
    
    // 四个方向扩散
    dfs(grid, i - 1, j);  // 上
    dfs(grid, i + 1, j);  // 下
    dfs(grid, i, j - 1);  // 左
    dfs(grid, i, j + 1);  // 右
}
```

## 边界 Case

**Case 1: 空网格**
```
grid = []
返回0 ✓
```

**Case 2: 全是水**
```
000
000

没有1，返回0 ✓
```

**Case 3: 全是陆地**
```
11
11

(0,0)=1，DFS把4个都变成0
count=1
返回1 ✓
```

**Case 4: 多个独立岛屿**
```
10001
01010
00100

(0,0)=1 → DFS → count=1
(0,4)=1 → DFS → count=2
(1,2)=1 → DFS → count=3
返回3 ✓
```

**Case 5: 对角线不算连通**
```
10
01

(0,0)=1 → DFS → count=1
(1,1)=1 → DFS → count=2

注意：对角线不算相邻，是2个岛屿 ✓
```
