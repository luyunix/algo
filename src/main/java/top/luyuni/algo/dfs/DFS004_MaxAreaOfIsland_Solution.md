# DFS004 - 岛屿的最大面积 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：给定二维网格，1表示陆地，0表示水，求最大岛屿面积

示例：
11000
11000
00100
00011

岛屿1（左上）：4个1
岛屿2（中间）：1个1
岛屿3（右下）：2个1

最大面积：4
```

### 第二步：发现规律

```
关键洞察：
- 和"岛屿数量"类似，需要找连通块
- 但不仅要计数，还要计算每个岛屿的面积
- 面积 = 连通块中1的个数

算法：
1. 遍历每个格子
2. 遇到1，开始DFS计算面积
3. DFS把连通的1变成0，同时计数
4. 更新最大面积
```

### 第三步：设计算法

```
主函数：
- 遍历网格
- 遇到1，调用dfs计算面积
- 更新maxArea

dfs函数：
- 返回当前连通块的面积
- 当前格子变成0
- 向四个方向扩散
- 返回 1 + 四个方向的面积之和
```

## 解法：网格DFS求面积

```java
public int maxAreaOfIsland(int[][] grid) {
    if (grid == null || grid.length == 0) return 0;
    
    int maxArea = 0;
    int m = grid.length, n = grid[0].length;
    
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (grid[i][j] == 1) {
                int area = dfs(grid, i, j);
                maxArea = Math.max(maxArea, area);
            }
        }
    }
    
    return maxArea;
}

private int dfs(int[][] grid, int i, int j) {
    int m = grid.length, n = grid[0].length;
    
    // 越界或不是陆地
    if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] != 1) {
        return 0;
    }
    
    // 标记为已访问
    grid[i][j] = 0;
    
    // 当前格子面积1 + 四个方向的面积
    return 1 + dfs(grid, i - 1, j)   // 上
             + dfs(grid, i + 1, j)   // 下
             + dfs(grid, i, j - 1)   // 左
             + dfs(grid, i, j + 1);  // 右
}
```

## 边界 Case

**Case 1: 空网格**
```
grid = []
返回0 ✓
```

**Case 2: 全是水**
```
000
000
返回0 ✓
```

**Case 3: 全是陆地**
```
11
11

一个岛屿，面积=4 ✓
```

**Case 4: 单个陆地**
```
00100
00000

只有一个1，面积=1 ✓
```

**Case 5: 多个相同面积的岛屿**
```
110011
110011
000000

两个岛屿，面积都是4，返回4 ✓
```
