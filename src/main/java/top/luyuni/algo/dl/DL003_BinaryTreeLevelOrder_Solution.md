# DL003 - 二叉树的层序遍历 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：按层遍历二叉树，每层节点放在一起

示例树：
    3
   / \
  9  20
    /  \
   15   7

层序遍历：
- 第1层：[3]
- 第2层：[9, 20]
- 第3层：[15, 7]

输出：[[3], [9,20], [15,7]]
```

### 第二步：发现规律

```
关键洞察：
- 访问第1层节点3时，把它的子节点9,20记下来
- 访问第2层节点9时，把它的子节点记下来（9没有子节点）
- 访问第2层节点20时，把它的子节点15,7记下来
- 访问第3层节点15,7...

发现规律：
- 先访问的节点，它的子节点也先被访问
- 这就是先进先出（FIFO）！
- 用队列实现
```

### 第三步：设计算法

```
BFS算法步骤：
1. 根节点入队
2. 当队列不为空：
   a. 当前层节点数 = 队列大小
   b. 循环当前层节点数次：
      - 出队一个节点
      - 记录节点值
      - 左子节点入队（如果存在）
      - 右子节点入队（如果存在）
   c. 当前层结果加入最终结果

关键点：
- 每层开始前，队列大小就是当前层的节点数
- 因为上一层节点的子节点都在队尾
```

---

## 解法：BFS + 队列

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;
    
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    
    while (!queue.isEmpty()) {
        int levelSize = queue.size();  // 当前层节点数
        List<Integer> level = new ArrayList<>();
        
        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        
        result.add(level);
    }
    
    return result;
}
```

## 边界 Case

**Case 1: 空树**

```
root = null
直接返回空列表 [] ✓
```

**Case 2: 只有根节点**

```
    1

队列: [1]
第1层：出队1，无子节点，队列[]
返回 [[1]] ✓
```

**Case 3: 只有左子树**

```
    1
   /
  2
 /
3

第1层: [1]，子节点2入队
第2层: [2]，子节点3入队
第3层: [3]，无子节点
返回 [[1], [2], [3]] ✓
```

**Case 4: 只有右子树**

```
1
 \
  2
   \
    3

第1层: [1]，子节点2入队
第2层: [2]，子节点3入队
第3层: [3]，无子节点
返回 [[1], [2], [3]] ✓
```

**Case 5: 满二叉树**

```
    1
   / \
  2   3
 / \ / \
4  5 6  7

第1层: [1]，子2,3入队
第2层: [2,3]，子4,5,6,7入队
第3层: [4,5,6,7]
返回 [[1], [2,3], [4,5,6,7]] ✓
```

## 记忆口诀

```
层序遍历用队列，根节点先入队
每层先算节点数，循环处理当前层
出队记录节点值，子节点再入队
一层结束加结果，队空结束返回答
```
