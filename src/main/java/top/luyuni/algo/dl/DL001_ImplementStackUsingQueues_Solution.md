# DL001 - 用队列实现栈 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：只用队列实现栈的4个操作
- push: 入栈
- pop: 出栈（移除并返回栈顶）
- top: 看栈顶
- empty: 判空

队列是FIFO（先进先出）
栈是LIFO（后进先出）

如何用FIFO实现LIFO？
```

### 第二步：发现关键

```
关键洞察：
- 队列只能从头出，但栈要从尾出
- 需要把队尾元素变到队头

方案：
- push时直接入队
- pop时，把前n-1个元素移到另一个队列，弹出最后一个
- top类似，只是不移除
```

### 第三步：优化方案

```
优化：只需要一个主队列，一个辅助队列

push(x):
- 直接入队

pop():
- 把主队列前size-1个元素移到辅助队列
- 弹出主队列最后一个元素
- 交换主辅队列

top():
- 类似pop，但要把弹出的元素再入队
```

---

## 解法：双队列

```java
class MyStack {
    private Queue<Integer> mainQueue;    // 主队列
    private Queue<Integer> helpQueue;    // 辅助队列
    
    public MyStack() {
        mainQueue = new LinkedList<>();
        helpQueue = new LinkedList<>();
    }
    
    // 直接入队
    public void push(int x) {
        mainQueue.offer(x);
    }
    
    // 把前n-1个移到辅助队列，弹出最后一个
    public int pop() {
        // 前n-1个移到辅助队列
        while (mainQueue.size() > 1) {
            helpQueue.offer(mainQueue.poll());
        }
        // 弹出最后一个（栈顶）
        int result = mainQueue.poll();
        // 交换
        Queue<Integer> temp = mainQueue;
        mainQueue = helpQueue;
        helpQueue = temp;
        return result;
    }
    
    public int top() {
        // 类似pop，但要放回去
        while (mainQueue.size() > 1) {
            helpQueue.offer(mainQueue.poll());
        }
        int result = mainQueue.poll();
        helpQueue.offer(result);  // 放回去
        // 交换
        Queue<Integer> temp = mainQueue;
        mainQueue = helpQueue;
        helpQueue = temp;
        return result;
    }
    
    public boolean empty() {
        return mainQueue.isEmpty();
    }
}
```

## 边界 Case

**Case 1: 连续push后连续pop**
```
push(1), push(2), push(3)
pop(), pop(), pop()

push后: 主[1,2,3]
pop1: 移动1,2到辅，弹出3，交换 → 主[1,2]
pop2: 移动1到辅，弹出2，交换 → 主[1]
pop3: 直接弹出1

结果: 3, 2, 1（符合栈LIFO）✓
```

**Case 2: push和pop交替**
```
push(1), pop(), push(2), top()

push(1): 主[1]
pop(): 弹出1，主[]
push(2): 主[2]
top(): 返回2 ✓
```

**Case 3: 空栈操作**
```
empty() → true
top() 或 pop() 应该抛出异常或返回特定值
```
