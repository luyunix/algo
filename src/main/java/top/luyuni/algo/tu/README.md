# TU 目录 - 图算法专题

这个目录收录图算法相关的题目。

## 为什么图算法不常考但重要？

图算法在面试中出现频率不如数组、链表、树那么高，但：

- 实际工程中经常遇到（依赖检测、路由算法、社交网络等）
- 一旦考到，能区分出候选人的算法功底
- 理解图算法有助于理解很多底层系统（如 Kubernetes 的依赖调度）

## 核心算法

| 算法       | 用途         | 典型题目       |
|----------|------------|------------|
| DFS/BFS  | 遍历、连通性     | 岛屿数量、课程表   |
| 拓扑排序     | 依赖检测、任务调度  | 课程表II、编译顺序 |
| 并查集      | 连通分量、最小生成树 | 省份数量、冗余连接  |
| Dijkstra | 最短路径       | 网络延迟时间     |
| 环检测      | 死锁检测       | 服务调用环检测    |

## 图算法学习建议

1. **先掌握存储方式**：邻接表 vs 邻接矩阵
2. **再掌握遍历**：DFS（递归）vs BFS（队列）
3. **最后掌握应用**：拓扑排序、最短路径、环检测
