# ZHAN001 - 有效的括号 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：判断括号字符串是否有效
示例：
- "()" 有效
- "()[]{}" 有效
- "(]" 无效（类型不匹配）
- "([)]" 无效（顺序不对）

有效条件：
1. 左括号必须有对应的右括号
2. 左括号必须先出现，右括号后闭合
3. 最近的左括号必须先闭合（栈的特性！）
```

### 第二步：发现栈的特性

```
关键洞察：括号匹配符合"后进先出"

示例："{[()]}"
- 遇到 {，等待匹配
- 遇到 [，等待匹配（[比{后遇到，要先闭合）
- 遇到 (，等待匹配（(比[后遇到，要先闭合）
- 遇到 )，匹配(
- 遇到 ]，匹配[
- 遇到 }，匹配{

发现规律：
- 左括号入栈（等待匹配）
- 右括号出栈匹配（必须匹配最近的左括号）
- 这就是栈的后进先出！
```

### 第三步：设计匹配规则

```
遇到左括号 ( [ { ：入栈

遇到右括号 ) ] } ：
1. 栈必须不为空（否则没有左括号匹配）
2. 栈顶必须是对应的左括号
3. 匹配成功，弹出栈顶

最后：栈必须为空（所有左括号都匹配了）
```

---

## 解法：栈匹配

```java
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') {
            // 左括号入栈
            stack.push(c);
        } else {
            // 右括号，需要匹配
            if (stack.isEmpty()) return false;
            
            char top = stack.pop();
            if (c == ')' && top != '(') return false;
            if (c == ']' && top != '[') return false;
            if (c == '}' && top != '{') return false;
        }
    }
    
    return stack.isEmpty();
}
```

**示例1："{[()]}"（有效）**

```

栈: ['{']

│  左括号，入栈                            │

栈: ['[', '{']

第3轮 c='(':
┌─────────────────────────────────────────┐
│  左括号，入栈                            │

栈: ['[', '{']

第5轮 c=']':
┌─────────────────────────────────────────┐
│  右括号，出栈匹配                        │
│  栈顶='['，匹配']' ✓                     │

└─────────────────────────────────────────┘
栈: ['{']

第6轮 c='}':
```

**示例2："([)]"（无效）**

```
初始：栈 = []

第1轮 c='(':
栈: ['(']

第2轮 c='[':
栈: ['[', '(']

第3轮 c=')':
┌─────────────────────────────────────────┐
│  右括号，出栈匹配                        │
│  栈顶='['，但')'需要匹配'('              │
│  '[' != '('，不匹配！                    │
└─────────────────────────────────────────┘
返回 false ✗
```

**示例3："(("（无效）**

```
初始：栈 = []

第1轮 c='(':
栈: ['(']

第2轮 c='(':
栈: ['(', '(']

结束，栈不为空，返回 false ✗
```

## 边界 Case

**Case 1: 空字符串**
```
s = ""
栈为空，返回 true ✓（空字符串视为有效）
```

**Case 2: 只有一个左括号**
```
s = "("
栈: ['(']，结束不为空，false ✗
```

**Case 3: 只有一个右括号**
```
s = ")"
遇到右括号，栈空，无法匹配，false ✗
```

**Case 4: 类型不匹配**
```
s = "(]"
'('入栈，']'出栈，'('!=']'，false ✗
```

**Case 5: 顺序不对**
```
s = "([)]"
见示例2，false ✗
```

```
遍历结束看栈空，空则有效非空假
```
