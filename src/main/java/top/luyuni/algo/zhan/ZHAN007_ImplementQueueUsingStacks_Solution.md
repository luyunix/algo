# ZHAN007 - 用栈实现队列 题解

## 思路是怎么想出来的

### 第一步：明确问题

```
问题：只用栈实现队列的4个操作
- push: 入队（队尾）
- pop: 出队（队头）
- peek: 看队头
- empty: 判空

栈是LIFO（后进先出）
队列是FIFO（先进先出）

如何用LIFO实现FIFO？
```

### 第二步：发现关键

```
关键洞察：
- 栈是反的，队列是正的
- 两个栈叠在一起，负负得正！

示例：
入队顺序：1, 2, 3

用栈A存储：[1, 2, 3]（3在栈顶）
倒到栈B：[3, 2, 1]（1在栈顶）

从栈B弹出：1, 2, 3（FIFO！）
```

### 第三步：设计双栈

```
输入栈（inStack）：专门接收push操作
输出栈（outStack）：专门处理pop和peek操作

push(x)：
- 直接压入输入栈

pop()：
- 如果输出栈为空，把输入栈全部倒到输出栈
- 从输出栈弹出

peek()：
- 类似pop，只是不移除

关键：只有当输出栈为空时才倒，保证顺序正确
```

---

## 解法：双栈（输入栈 + 输出栈）

```java
class MyQueue {
    private Deque<Integer> inStack;   // 输入栈
    private Deque<Integer> outStack;  // 输出栈
    
    public MyQueue() {
        inStack = new ArrayDeque<>();
        outStack = new ArrayDeque<>();
    }
    
    // 直接入输入栈
    public void push(int x) {
        inStack.push(x);
    }
    
    // 从输出栈出，空则先倒栈
    public int pop() {
        peek();  // 确保输出栈有元素
        return outStack.pop();
    }
    
    public int peek() {
        if (outStack.isEmpty()) {
            // 输入栈全部倒到输出栈
            while (!inStack.isEmpty()) {
                outStack.push(inStack.pop());
            }
        }
        return outStack.peek();
    }
    
    public boolean empty() {
        return inStack.isEmpty() && outStack.isEmpty();
    }
}
```

## 关键点分析

### 为什么倒栈只在输出栈为空时做？

```
反例：如果每次pop都倒栈

push(1), push(2), pop(), pop()

第一次pop：输入[2,1] → 倒到输出[1,2] → 弹出1
第二次pop：输出[2]，但如果我们又倒栈...
输入[2] → 倒到输出[2] → 弹出2

看起来是对的？再看这个：
push(1), push(2), pop(), push(3), pop()

第一次pop：输入[2,1] → 倒[1,2] → 弹出1
push(3)：输入[3]
第二次pop（错误做法）：输入[3] → 倒[3] → 弹出3

但正确结果应该是2！

结论：必须等输出栈空了再倒，保证顺序
```

### 复杂度分析

```
push: O(1)
peek/pop: 均摊O(1)
  - 每个元素最多被压入输入栈1次
  - 最多被转移到输出栈1次
  - 最多被弹出1次
  - 总共3次操作，均摊O(1)
```

## 边界 Case

**Case 1: 连续push后连续pop**
```
push(1), push(2), push(3)
pop(), pop(), pop()

push后: 输入[3,2,1]
pop1: 倒栈→输出[1,2,3]，弹出1，输出[2,3]
pop2: 直接弹出2，输出[3]
pop3: 直接弹出3，输出[]

结果: 1, 2, 3（符合FIFO）✓
```

**Case 2: push和pop交替**
```
push(1), pop(), push(2), pop()

push(1): 输入[1]
pop(): 倒栈→输出[1]，弹出1
push(2): 输入[2]
pop(): 输出空，倒栈→输出[2]，弹出2

结果: 1, 2 ✓
```

**Case 3: 空队列操作**
```
empty() → true
peek() 或 pop() 应该抛出异常
```

## 与"用队列实现栈"对比

```
用队列实现栈：              用栈实现队列：
- 一个主队列，一个辅助队列    - 一个输入栈，一个输出栈
- pop时倒腾n-1个元素         - pop时可能倒腾全部元素
- 每次pop都要倒腾            - 只有输出栈空时才倒腾
- 时间复杂度高               - 均摊O(1)，更优

结论：栈实现队列比队列实现栈更高效！
```
