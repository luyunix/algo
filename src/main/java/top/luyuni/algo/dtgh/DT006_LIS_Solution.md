# DT006 - 最长递增子序列 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：找最长的严格递增子序列（不要求连续）
示例：nums = [10, 9, 2, 5, 3, 7, 101, 18]
答案：4（子序列 [2, 3, 7, 101]）

暴力法：枚举所有子序列，检查是否递增
- 子序列数量：O(2^n)
- 需要DP优化
```

### 第二步：尝试直接定义（失败）

```
尝试：dp[i] = 前i个元素的最长递增子序列长度

问题：无法递推！
- 前5个的最长长度是2（假设）
- 第6个元素是7
- 7能不能接在前面？不知道！
- 因为不知道前面的子序列结尾是什么

信息不足：只知道长度2，但不知道结尾元素是多少
```

### 第三步：关键洞察——固定结尾位置

```
核心思想：子序列问题必须固定结尾位置！

新定义：dp[i] = 以第i个元素结尾的最长递增子序列长度

为什么这样可以？
- 结尾固定在i，值就是nums[i]
- 对于j < i，如果nums[j] < nums[i]，就能接上去
- dp[i] = max(dp[j] + 1) for all j < i and nums[j] < nums[i]

验证：
nums = [10, 9, 2, 5, 3, 7]
计算dp[5]（以7结尾）：
- j=0: 10 < 7? 否
- j=1: 9 < 7? 否
- j=2: 2 < 7? 是，dp[2]+1 = 2
- j=3: 5 < 7? 是，dp[3]+1 = 3
- j=4: 3 < 7? 是，dp[4]+1 = 3
- dp[5] = 3 ✓
```

### 第四步：如何从dp得到答案

```
dp[i] 是以i结尾的最长长度，但原问题要的是全局最长

最终答案 = max(dp[0], dp[1], ..., dp[n-1])

示例：
dp = [1, 1, 1, 2, 2, 3, 4, 4]
max(dp) = 4 ✓
```

## 核心模型：子序列DP

子序列问题关键：**以某个位置结尾定义状态**

---

## 递推公式推导

**状态定义**：`dp[i] = 以 nums[i] 结尾的最长递增子序列长度`

**思考**：怎么得到以 i 结尾的最长序列？

```
nums = [10, 9, 2, 5, 3, 7, 101, 18]

计算 dp[5]（以7结尾）：
- 看前面比7小的元素：2, 5, 3
- 以2结尾的序列长度：dp[2]=1，接7变成2
- 以5结尾的序列长度：dp[3]=2，接7变成3
- 以3结尾的序列长度：dp[4]=2，接7变成3
- 取最大：dp[5] = 3
```

通用公式：
```
dp[i] = max(dp[j] + 1) for all j < i and nums[j] < nums[i]
```

## 代码实现

### 基础DP O(n²)

```java
public int lengthOfLIS(int[] nums) {
    int n = nums.length;
    int[] dp = new int[n];
    Arrays.fill(dp, 1);  // 每个元素至少可以单独成序列
    
    int maxLen = 1;
    
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
            }
        }
        maxLen = Math.max(maxLen, dp[i]);
    }
    
    return maxLen;
}
```

### 二分优化 O(n log n)

```java
public int lengthOfLISBinary(int[] nums) {
    // tails[i] = 长度为i+1的递增子序列的最小末尾元素
    int[] tails = new int[nums.length];
    int size = 0;
    
    for (int num : nums) {
        // 二分查找：第一个 >= num 的位置
        int left = 0, right = size;
        while (left < right) {
            int mid = left + (right - left) / 2;
            if (tails[mid] < num) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        
        tails[left] = num;
        if (left == size) size++;
    }
    
    return size;
}
```

## 边界 Case

**Case 1: 全部递减**
```
nums = [5, 4, 3, 2, 1]
dp = [1, 1, 1, 1, 1]
返回 1 ✓
每个元素单独成序列
```

**Case 2: 全部递增**
```
nums = [1, 2, 3, 4, 5]
dp = [1, 2, 3, 4, 5]
返回 5 ✓
整个数组就是LIS
```

**Case 3: 重复元素**
```
nums = [1, 1, 1, 1]
dp = [1, 1, 1, 1]
返回 1 ✓
严格递增，重复不算
```

**Case 4: 单个元素**
```
nums = [7]
返回 1 ✓
```

## DP 四步法

| 步骤 | 内容 | 本题 |
|------|------|------|
| 1. 定义状态 | dp[i] 表示什么 | dp[i] = 以 nums[i] 结尾的最长递增子序列长度 |
| 2. 状态转移 | dp[i] 怎么算 | dp[i] = max(dp[j]+1) 对所有 j<i 且 nums[j]<nums[i] |
| 3. 初始条件 | dp[i] 初始值 | dp[i] = 1 (每个元素至少单独成序列) |
| 4. 遍历顺序 | 从哪到哪 | i: 0→n-1, j: 0→i-1 |

---

## 易错点详解

### 1. 子序列 vs 子数组

```
子数组（连续）：[2,5,3] 是子数组
子序列（不连续）：[2,3,7] 是子序列（跳过5）
```

### 2. 严格递增

条件是 `nums[j] < nums[i]`，不是 `<=`

### 3. 最后答案是 max(dp)

```java
// 错误
return dp[n-1];

// 正确
return maxLen;  // 记录遍历过程中的最大值
```
