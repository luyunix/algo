# DT010 - 打家劫舍 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：偷房屋使总金额最大，相邻房屋不能同时偷
示例：nums = [2, 7, 9, 3, 1]
答案：12（偷房屋1、3、5：2+9+1=12）

暴力法：枚举所有不相邻的子集
- 子集数量：O(2^n)
- 需要DP
```

### 第二步：尝试直接定义

```
问题所求：最大金额
尝试：dp[i] = 前i个房屋能偷的最大金额

验证是否可递推：
- 考虑第i个房屋，有两种选择：偷或不偷
- 不偷：金额 = dp[i-1]
- 偷：金额 = dp[i-2] + nums[i-1]（不能偷i-1）
- dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])

可以递推！✓
```

### 第三步：理解为什么需要dp[i-2]

```
关键约束：相邻不能同时偷

如果偷第i个：
- 第i-1个绝对不能偷
- 只能从前i-2个中选最优方案
- 所以是 dp[i-2] + nums[i-1]

如果不偷第i个：
- 第i-1个可以偷也可以不偷
- 直接继承 dp[i-1]
```

### 第四步：确定初始条件

```
dp[0] = 0（0个房屋，金额为0）
dp[1] = nums[0]（只有1个房屋，偷它）

答案：dp[n]
```

## 核心模型：偷或不偷

相邻房屋不能同时偷，这是一个约束条件。

---

## 递推公式推导

**状态定义**：`dp[i] = 偷窃前i个房屋能获得的最高金额`

**考虑第i个房屋**：

```
情况1：不偷第i个
       最高金额 = dp[i-1]（前i-1个房屋的最高金额）

情况2：偷第i个
       因为不能偷相邻的，第i-1个不能偷
       最高金额 = dp[i-2] + nums[i-1]（前i-2个的最高金额 + 当前金额）

dp[i] = max(不偷, 偷) = max(dp[i-1], dp[i-2] + nums[i-1])
```

## 代码实现

### 基础DP

```java
public int rob(int[] nums) {
    int n = nums.length;
    if (n == 0) return 0;
    if (n == 1) return nums[0];
    
    int[] dp = new int[n + 1];
    dp[0] = 0;
    dp[1] = nums[0];
    
    for (int i = 2; i <= n; i++) {
        dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i - 1]);
    }
    
    return dp[n];
}
```

### 空间优化

```java
public int robOptimized(int[] nums) {
    int n = nums.length;
    if (n == 0) return 0;
    if (n == 1) return nums[0];
    
    int prev2 = 0;  // dp[i-2]
    int prev1 = nums[0];  // dp[i-1]
    
    for (int i = 2; i <= n; i++) {
        int curr = Math.max(prev1, prev2 + nums[i - 1]);
        prev2 = prev1;
        prev1 = curr;
    }
    
    return prev1;
}
```

### 环形版本（打家劫舍II）

```java
public int robCircular(int[] nums) {
    int n = nums.length;
    if (n == 0) return 0;
    if (n == 1) return nums[0];
    
    // 情况1：不偷第一个房屋
    int case1 = robRange(nums, 1, n - 1);
    // 情况2：不偷最后一个房屋
    int case2 = robRange(nums, 0, n - 2);
    
    return Math.max(case1, case2);
}

private int robRange(int[] nums, int start, int end) {
    int prev2 = 0;
    int prev1 = 0;
    
    for (int i = start; i <= end; i++) {
        int curr = Math.max(prev1, prev2 + nums[i]);
        prev2 = prev1;
        prev1 = curr;
    }
    
    return prev1;
}
```

## 完整执行图解

**输入**: nums = [2, 7, 9, 3, 1]

```
初始化：
dp[0] = 0      // 0个房屋，金额为0
dp[1] = 2      // 第1个房屋，金额2

第1轮 i=2（第2个房屋，金额7）：
┌─────────────────────────────────────────┐
│  选择1：不偷第2个房屋                    │
│          金额 = dp[1] = 2               │
│                                         │
│  选择2：偷第2个房屋                      │
│          金额 = dp[0] + nums[1]         │
│                = 0 + 7 = 7              │
│          （不能偷第1个，相邻）           │
│                                         │
│  dp[2] = max(2, 7) = 7                  │
│  决策：偷第2个房屋！                     │
└─────────────────────────────────────────┘
dp = [0, 2, 7]

第2轮 i=3（第3个房屋，金额9）：
┌─────────────────────────────────────────┐
│  选择1：不偷第3个房屋                    │
│          金额 = dp[2] = 7               │
│                                         │
│  选择2：偷第3个房屋                      │
│          金额 = dp[1] + nums[2]         │
│                = 2 + 9 = 11             │
│          （不能偷第2个，偷第1个）        │
│                                         │
│  dp[3] = max(7, 11) = 11                │
│  决策：偷第3个房屋！                     │
└─────────────────────────────────────────┘
dp = [0, 2, 7, 11]

第3轮 i=4（第4个房屋，金额3）：
┌─────────────────────────────────────────┐
│  选择1：不偷第4个房屋                    │
│          金额 = dp[3] = 11              │
│                                         │
│  选择2：偷第4个房屋                      │
│          金额 = dp[2] + nums[3]         │
│                = 7 + 3 = 10             │
│          （不能偷第3个，偷第2个）        │
│                                         │
│  dp[4] = max(11, 10) = 11               │
│  决策：不偷第4个房屋！                   │
└─────────────────────────────────────────┘
dp = [0, 2, 7, 11, 11]

第4轮 i=5（第5个房屋，金额1）：
┌─────────────────────────────────────────┐
│  选择1：不偷第5个房屋                    │
│          金额 = dp[4] = 11              │
│                                         │
│  选择2：偷第5个房屋                      │
│          金额 = dp[3] + nums[4]         │
│                = 11 + 1 = 12            │
│          （不能偷第4个，偷第3个）        │
│                                         │
│  dp[5] = max(11, 12) = 12               │
│  决策：偷第5个房屋！                     │
└─────────────────────────────────────────┘
dp = [0, 2, 7, 11, 11, 12]

返回 dp[5] = 12 ✓

**完整 DP 表：**

| i | 0 | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|---|
| 房屋 | - | 1 | 2 | 3 | 4 | 5 |
| 金额 | - | 2 | 7 | 9 | 3 | 1 |
| dp[i] | 0 | 2 | 7 | 11 | 11 | 12 |
| 决策 | - | 偷 | 偷 | 偷 | 不偷 | 偷 |

最优方案：偷房屋2(7) + 房屋3(9) + 房屋5(1) = 不对
等等，重新检查：
- 偷房屋2(7)就不能偷房屋1和3
- dp[3]=11是偷房屋1(2)+房屋3(9)=11 ✓
- dp[5]=12是dp[3]+房屋5(1)=11+1=12 ✓

最终方案：偷房屋1(2) + 房屋3(9) + 房屋5(1) = 12 ✓
```

第3轮 i=4（金额3）：

第4轮 i=5（金额1）：

## 边界 Case

**Case 1: 空数组**
```
nums = []
返回 0 ✓
```

**Case 2: 只有一个房屋**
```
nums = [5]
返回 5 ✓
```

**Case 3: 两个房屋**
```
nums = [3, 5]
只能偷一个，选大的
dp[2] = max(dp[1], dp[0]+5) = max(3, 5) = 5
返回 5 ✓
```

**Case 4: 金额递增**
```
nums = [1, 2, 3, 4, 5]
最优：偷2+4=6 或 1+3+5=9
返回 9 ✓
```

**Case 5: 金额递减**
```
nums = [5, 4, 3, 2, 1]
最优：偷5+3+1=9
返回 9 ✓
```

## DP 四步法

| 步骤 | 内容 | 本题 |
|------|------|------|
| 1. 定义状态 | dp[i] 表示什么 | dp[i] = 偷前 i 个房屋的最大金额 |
| 2. 状态转移 | dp[i] 怎么算 | max(不偷，偷) = max(dp[i-1], dp[i-2]+nums[i-1]) |
| 3. 初始条件 | dp[0], dp[1] | dp[0]=0, dp[1]=nums[0] |
| 4. 遍历顺序 | 从哪到哪 | i: 2→n |

---

## 和爬楼梯的对比

| | 爬楼梯 | 打家劫舍 |
|--|--------|----------|
| 递推公式 | dp[i] = dp[i-1] + dp[i-2] | dp[i] = max(dp[i-1], dp[i-2] + nums[i-1]) |
| 含义 | 两种选择都要 | 两种选择取max |
| 目标 | 计数 | 最值 |

## 环形版本思路

```
房屋是环形：0-1-2-3-4-0

如果偷了0，就不能偷4
如果偷了4，就不能偷0

分两种情况：
1. 不偷0：在[1, n-1]范围内求最大值
2. 不偷n-1：在[0, n-2]范围内求最大值

答案是两种情况的最大值
```
