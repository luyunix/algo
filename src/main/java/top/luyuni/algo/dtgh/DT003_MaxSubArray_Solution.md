# DT003 - 最大子数组和 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：找和最大的连续子数组
示例：nums = [-2, 1, -3, 4, -1, 2, 1]
答案：[4, -1, 2, 1]，和为 6

暴力法：枚举所有子数组，求最大和
- 子数组数量：O(n²)
- 需要更优的解法
```

### 第二步：尝试直接定义（失败）

```
尝试：dp[i] = 前i个元素的最大子数组和

问题：无法递推！
- 前5个的最大子数组和是7（假设）
- 第6个元素是-1
- 这个-1要不要加入？不知道！
- 因为不知道前面的子数组在哪结束

信息不足，需要更精确的状态定义
```

### 第三步：调整定义使其可递推

```
关键洞察：必须固定子数组的结尾位置！

新定义：dp[i] = 以第i个元素结尾的最大子数组和

为什么这样可以？
- 结尾固定在i，就只有两种选择：
  1. 只选nums[i]自己（重新开始）
  2. 接在以i-1结尾的子数组后面（延续）

递推公式：
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

### 第四步：如何从dp得到答案

```
dp[i] 是以i结尾的最大和，但原问题要的是全局最大

所以最终答案 = max(dp[0], dp[1], ..., dp[n-1])

示例：
nums = [-2, 1, -3, 4, -1, 2, 1]
dp   = [-2, 1, -2, 4,  3, 5, 6]

max(dp) = 6 ✓
```

## 核心难点：如何定义状态

这道题的关键在于：**状态定义要包含位置信息**

---

## 状态定义的思考过程

**错误尝试**：`dp[i] = 前i个元素的最大子数组和`
- 问题：不知道最大子数组在哪结束，无法递推

**正确做法**：`dp[i] = 以第i个元素结尾的最大子数组和`
- 好处：位置确定了，可以递推

---

## 递推公式的推导

以第 i 个元素结尾的子数组，只有两种可能：

```
情况1：只包含nums[i]自己
       [..., nums[i]]
            ↑
           从这里开始新的子数组

情况2：接在以i-1结尾的子数组后面
       [..., nums[i-1], nums[i]]
            ↑            ↑
           dp[i-1]     加上nums[i]
```

所以：
```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

## 代码实现

### 基础DP

```java
public int maxSubArray(int[] nums) {
    int n = nums.length;
    int[] dp = new int[n];
    dp[0] = nums[0];
    
    int maxSum = dp[0];
    
    for (int i = 1; i < n; i++) {
        dp[i] = Math.max(nums[i], dp[i-1] + nums[i]);
        maxSum = Math.max(maxSum, dp[i]);
    }
    
    return maxSum;
}
```

### Kadane算法（空间优化）

```java
public int maxSubArrayKadane(int[] nums) {
    int maxSum = nums[0];
    int currSum = nums[0];  // 以当前元素结尾的最大和
    
    for (int i = 1; i < nums.length; i++) {
        // 要么重新开始，要么延续前面
        currSum = Math.max(nums[i], currSum + nums[i]);
        // 更新全局最大
        maxSum = Math.max(maxSum, currSum);
    }
    
    return maxSum;
}
```

## 完整执行图解

**输入**: nums = [-2, 1, -3, 4, -1, 2, 1]

```
初始化：
dp[0] = nums[0] = -2
maxSum = -2

第1轮 i=1, nums[1]=1:
┌─────────────────────────────────────────┐
│  选择1：只选nums[1] = 1                  │
│  选择2：接在dp[0]后面 = -2 + 1 = -1      │
│  dp[1] = max(1, -1) = 1                 │
│  解释：从1重新开始更优                   │
└─────────────────────────────────────────┘

dp = [-2, 1, 0, 0, 0, 0, 0]
maxSum = max(-2, 1) = 1

第2轮 i=2, nums[2]=-3:
┌─────────────────────────────────────────┐
│  选择1：只选nums[2] = -3                 │
│  选择2：接在dp[1]后面 = 1 + (-3) = -2    │
│  dp[2] = max(-3, -2) = -2               │
│  解释：虽然都是负数，但接在后面损失更小    │
└─────────────────────────────────────────┘

dp = [-2, 1, -2, 0, 0, 0, 0]
maxSum = max(1, -2) = 1

第3轮 i=3, nums[3]=4:
┌─────────────────────────────────────────┐
│  选择1：只选nums[3] = 4                  │
│  选择2：接在dp[2]后面 = -2 + 4 = 2       │
│  dp[3] = max(4, 2) = 4                  │
│  解释：从4重新开始更优                   │
└─────────────────────────────────────────┘

dp = [-2, 1, -2, 4, 0, 0, 0]
maxSum = max(1, 4) = 4

第4轮 i=4, nums[4]=-1:
┌─────────────────────────────────────────┐
│  选择1：只选nums[4] = -1                 │
│  选择2：接在dp[3]后面 = 4 + (-1) = 3     │
│  dp[4] = max(-1, 3) = 3                 │
│  解释：接在[4]后面，形成[4,-1]            │
└─────────────────────────────────────────┘

dp = [-2, 1, -2, 4, 3, 0, 0]
maxSum = max(4, 3) = 4

第5轮 i=5, nums[5]=2:
┌─────────────────────────────────────────┐
│  选择1：只选nums[5] = 2                  │
│  选择2：接在dp[4]后面 = 3 + 2 = 5        │
│  dp[5] = max(2, 5) = 5                  │
│  解释：接在[4,-1]后面，形成[4,-1,2]       │
└─────────────────────────────────────────┘

dp = [-2, 1, -2, 4, 3, 5, 0]
maxSum = max(4, 5) = 5

第6轮 i=6, nums[6]=1:
┌─────────────────────────────────────────┐
│  选择1：只选nums[6] = 1                  │
│  选择2：接在dp[5]后面 = 5 + 1 = 6        │
│  dp[6] = max(1, 6) = 6                  │
│  解释：接在[4,-1,2]后面，形成[4,-1,2,1]   │
└─────────────────────────────────────────┘

dp = [-2, 1, -2, 4, 3, 5, 6]
maxSum = max(5, 6) = 6

返回 maxSum = 6 ✓

**完整 DP 表：**

| i | 0 | 1 | 2 | 3 | 4 | 5 | 6 |
|---|---|---|---|---|---|---|---|
| nums | -2 | 1 | -3 | 4 | -1 | 2 | 1 |
| dp[i] | -2 | 1 | -2 | 4 | 3 | 5 | 6 |
| 子数组 | [-2] | [1] | [-3] | [4] | [4,-1] | [4,-1,2] | [4,-1,2,1] |

最终结果：子数组 [4, -1, 2, 1]，和为 6
```

## 边界 Case

**Case 1: 全负数**
```
nums = [-5, -2, -8, -1]

dp[0] = -5, maxSum = -5
dp[1] = max(-2, -5-2) = -2, maxSum = -2
dp[2] = max(-8, -2-8) = -8, maxSum = -2
dp[3] = max(-1, -8-1) = -1, maxSum = -1

返回 -1（最大的那个负数）✓
```

**Case 2: 全正数**
```
nums = [1, 2, 3, 4]

dp[0] = 1
dp[1] = max(2, 1+2) = 3
dp[2] = max(3, 3+3) = 6
dp[3] = max(4, 6+4) = 10

返回 10（整个数组）✓
```

**Case 3: 单元素**
```
nums = [5]
返回 5 ✓
```

## DP 四步法

| 步骤 | 内容 | 本题 |
|------|------|------|
| 1. 定义状态 | dp[i] 表示什么 | dp[i] = 以第 i 个元素结尾的最大子数组和 |
| 2. 状态转移 | dp[i] 怎么算 | dp[i] = max(nums[i], dp[i-1] + nums[i]) |
| 3. 初始条件 | dp[0] | dp[0] = nums[0] |
| 4. 遍历顺序 | 从哪到哪 | 从 1 到 n-1 |

---

## 关键洞察

**为什么要"以第i个元素结尾"？**

因为子数组必须是连续的！
- 如果知道以 i-1 结尾的最大子数组
- 那么以 i 结尾的要么接在后面，要么从 i 重新开始
- 这样就能递推了

**如果题目改成"子序列"（不要连续）呢？**

那就简单了：所有正数相加就是答案！
