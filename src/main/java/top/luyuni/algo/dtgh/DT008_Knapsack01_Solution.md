# DT008 - 01背包 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：在容量限制下，选物品使总价值最大
约束：每个物品只能选0次或1次
示例：weight=[1,3,4], value=[15,20,30], W=4
答案：35（选物品0和物品1：1+3=4，价值15+20=35）

暴力法：枚举所有子集
- 子集数量：O(2^n)
- 需要DP
```

### 第二步：分析状态维度

```
问题涉及两个变量：
1. 考虑到第几个物品
2. 当前背包容量

自然想到二维状态：
dp[i][w] = 考虑前i个物品，容量为w时的最大价值

为什么需要两个维度？
- 物品维度：决定考虑哪些物品
- 容量维度：决定能装多少
- 两者相互制约，缺一不可
```

### 第三步：分类讨论（选或不选）

```
对于第i个物品，只有两种选择：

情况1：不选第i个物品
- 价值 = dp[i-1][w]（前i-1个物品，容量w的最大价值）

情况2：选第i个物品（前提：w >= weight[i-1]）
- 价值 = dp[i-1][w-weight[i-1]] + value[i-1]
- （前i-1个物品，剩余容量的最大价值 + 当前物品价值）

dp[i][w] = max(不选, 选)
```

### 第四步：确定初始条件

```
dp[0][w] = 0（0个物品，价值为0）
dp[i][0] = 0（容量为0，装不下任何东西）

答案：dp[n][W]
```

## 核心模型：选或不选

每个物品只有两种状态：选（1次）或不选（0次）。

---

## 递推公式推导

**状态定义**：`dp[i][w] = 考虑前i个物品，容量w时的最大价值`

**思考**：对于第i个物品，选还是不选？

```
不选：dp[i-1][w]（前i-1个物品，容量w的最大价值）

选：dp[i-1][w-weight[i-1]] + value[i-1]
    （前i-1个物品，剩余容量的最大价值 + 当前物品价值）

dp[i][w] = max(不选, 选)
```

## 代码实现

### 二维DP

```java
public int knapsack(int[] weight, int[] value, int W) {
    int n = weight.length;
    int[][] dp = new int[n + 1][W + 1];
    
    for (int i = 1; i <= n; i++) {
        for (int w = 1; w <= W; w++) {
            // 不选第i个物品
            dp[i][w] = dp[i - 1][w];
            
            // 选第i个物品（如果装得下）
            if (w >= weight[i - 1]) {
                dp[i][w] = Math.max(dp[i][w], 
                    dp[i - 1][w - weight[i - 1]] + value[i - 1]);
            }
        }
    }
    
    return dp[n][W];
}
```

### 空间优化（一维DP）

```java
public int knapsackOptimized(int[] weight, int[] value, int W) {
    int n = weight.length;
    int[] dp = new int[W + 1];
    
    for (int i = 0; i < n; i++) {
        // 关键：从大到小遍历！
        for (int w = W; w >= weight[i]; w--) {
            dp[w] = Math.max(dp[w], dp[w - weight[i]] + value[i]);
        }
    }
    
    return dp[W];
}
```

## 为什么一维要倒序遍历？

```
假设正在处理物品i，重量2，价值3

正序遍历（错误）：
w=2: dp[2] = max(dp[2], dp[0]+3) = 3
w=3: dp[3] = max(dp[3], dp[1]+3) = 3
w=4: dp[4] = max(dp[4], dp[2]+3) = 6  ← 用了更新后的dp[2]！

这样dp[2]已经是选了物品i后的值，dp[4]又用了它
相当于物品i被用了两次！这是完全背包！

倒序遍历（正确）：
w=4: dp[4] = max(dp[4], dp[2]+3) = 3  ← dp[2]还是上一行的值
w=3: dp[3] = max(dp[3], dp[1]+3) = 3
w=2: dp[2] = max(dp[2], dp[0]+3) = 3

每个物品只用一次！
```

## 完整执行图解

**输入**: weight = [1, 3, 4], value = [15, 20, 30], W = 4

```
初始化：
    0  1  2  3  4
0   0  0  0  0  0

第1轮 i=1（物品0：重量1，价值15）：
┌─────────────────────────────────────────┐
│  w=1: 1>=1 ✓                            │
│       不选：dp[0][1] = 0                │
│       选：dp[0][0] + 15 = 15            │
│       dp[1][1] = max(0, 15) = 15        │
│  w=2: 2>=1 ✓                            │
│       选：dp[0][1] + 15 = 15            │
│       dp[1][2] = 15                     │
│  w=3,4: 同理                            │
└─────────────────────────────────────────┘

    0  1  2  3  4
0   0  0  0  0  0
1   0 15 15 15 15
解释：容量>=1时都能装下物品0

第2轮 i=2（物品1：重量3，价值20）：
┌─────────────────────────────────────────┐
│  w=1: 1<3，装不下，不选                 │
│       dp[2][1] = dp[1][1] = 15          │
│  w=2: 2<3，装不下                       │
│       dp[2][2] = dp[1][2] = 15          │
│  w=3: 3>=3 ✓                            │
│       不选：dp[1][3] = 15               │
│       选：dp[1][0] + 20 = 20            │
│       dp[2][3] = max(15, 20) = 20       │
│  w=4: 4>=3 ✓                            │
│       不选：dp[1][4] = 15               │
│       选：dp[1][1] + 20 = 15+20 = 35    │
│       dp[2][4] = max(15, 35) = 35       │
│       ← 装物品0+物品1！                 │
└─────────────────────────────────────────┘

    0  1  2  3  4
0   0  0  0  0  0
1   0 15 15 15 15
2   0 15 15 20 35

第3轮 i=3（物品2：重量4，价值30）：
┌─────────────────────────────────────────┐
│  w=1,2,3: 都<4，装不下                  │
│       继承上一行：15, 15, 20            │
│  w=4: 4>=4 ✓                            │
│       不选：dp[2][4] = 35               │
│       选：dp[2][0] + 30 = 30            │
│       dp[3][4] = max(35, 30) = 35       │
│       ← 不选更优！物品0+1比物品2好      │
└─────────────────────────────────────────┘

**最终 DP 表：**

| i \ w | 0 | 1 | 2 | 3 | 4 |
|-------|---|---|---|---|---|
| 0 | 0 | 0 | 0 | 0 | 0 |
| 1 | 0 | 15 | 15 | 15 | 15 |
| 2 | 0 | 15 | 15 | 20 | 35 |
| 3 | 0 | 15 | 15 | 20 | 35 |

返回 dp[3][4] = 35 ✓
最优方案：装物品0(重1,价15) + 物品1(重3,价20) = 35
```

## 一维优化版执行图解

```
weight = [1, 3, 4], value = [15, 20, 30], W = 4

初始化：
dp = [0, 0, 0, 0, 0]

第1轮 i=0（物品0：重量1，价值15）：
┌─────────────────────────────────────────┐
│  倒序遍历 w=4,3,2,1：                    │
│  w=4: dp[4] = max(0, dp[3]+15) = 15     │
│  w=3: dp[3] = max(0, dp[2]+15) = 15     │
│  w=2: dp[2] = max(0, dp[1]+15) = 15     │
│  w=1: dp[1] = max(0, dp[0]+15) = 15     │
└─────────────────────────────────────────┘
dp = [0, 15, 15, 15, 15]

第2轮 i=1（物品1：重量3，价值20）：
┌─────────────────────────────────────────┐
│  倒序遍历 w=4,3：                        │
│  w=4: dp[4] = max(15, dp[1]+20)         │
│             = max(15, 15+20) = 35       │
│  w=3: dp[3] = max(15, dp[0]+20) = 20    │
│  w=2,1: <3，跳过                        │
└─────────────────────────────────────────┘
dp = [0, 15, 15, 20, 35]

第3轮 i=2（物品2：重量4，价值30）：
┌─────────────────────────────────────────┐
│  倒序遍历 w=4：                          │
│  w=4: dp[4] = max(35, dp[0]+30)         │
│             = max(35, 30) = 35          │
│       不更新，之前方案更优               │
└─────────────────────────────────────────┘
dp = [0, 15, 15, 20, 35]

返回 dp[4] = 35 ✓
```

## 边界 Case

**Case 1: 背包容量为0**
```
W = 0
什么都装不下
返回 0 ✓
```

**Case 2: 所有物品都装不下**
```
weight = [5, 6, 7], W = 4
所有物品重量 > W
返回 0 ✓
```

**Case 3: 只能装一个**
```
weight = [3, 4, 5], value = [10, 20, 15], W = 5
只能装物品2(重5)或物品1(重4)+...不行
最优：物品2，价值15？不对
等等，W=5，可以装物品0(重3,价10) + ？没有其他
或者物品2(重5,价15)
返回 15 ✓
```

**Case 4: 价值相同选轻的**
```
weight = [2, 3], value = [10, 10], W = 3
选物品0：重量2，价值10
选物品1：重量3，价值10
都可以，返回 10 ✓
```

## DP四步法

| 步骤 | 内容 |
|------|------|
| 1. 定义状态 | dp[i][w] = 前 i 个物品，容量 w 的最大价值 |
| 2. 状态转移 | max(不选，选) |
| 3. 初始条件 | dp[0][w]=0, dp[i][0]=0 |
| 4. 遍历顺序 | i 从 1 到 n，w 从 1 到 W |

---

## 01背包 vs 完全背包

| 类型 | 特点 | 一维遍历顺序 |
|------|------|-------------|
| 01背包 | 每个物品用1次 | 容量从大到小 |
| 完全背包 | 每个物品无限次 | 容量从小到大 |
