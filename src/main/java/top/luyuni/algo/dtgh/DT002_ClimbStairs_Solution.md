# DT002 - 爬楼梯 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：到第n阶有多少种不同的方法
约束：每次可以迈1级或2级

暴力法：枚举所有可能的走法
- n=3 时：1+1+1, 1+2, 2+1
- 共3种方法

问题所求：方法数
```

### 第二步：尝试直接定义

```
尝试：dp[i] = 到第i阶的方法数

验证这个定义是否可行：
- 可以递推吗？dp[i] 能从前面推出来吗？
- 可以！从最后一步倒推
```

### 第三步：从最后一步倒推（核心技巧）

```
关键：看最后一步是怎么走的

到第i阶，最后一步有两种可能：
┌─────────────────────────────────────────┐
│  情况1：从第i-1阶迈1级上来              │
│          前面有 dp[i-1] 种方法          │
│                                         │
│  情况2：从第i-2阶迈2级上来              │
│          前面有 dp[i-2] 种方法          │
└─────────────────────────────────────────┘

所以：dp[i] = dp[i-1] + dp[i-2]

为什么不是 dp[i] = dp[i-1] + 1？
- dp[i-1] 是方法数，不是步数！
- 每种方法再迈1级，还是那么多方法
```

### 第四步：验证无后效性

```
算 dp[3] 时：
- 只需要 dp[2] 和 dp[1] 的值
- 不需要知道 dp[2] 具体是哪2种方法
- 满足无后效性 ✓
```

## 核心技巧：从最后一步倒推

这是找递推公式的万能方法！

---

## 思考过程

**问题**：到达第 n 阶有多少种方法？

**倒推**：最后一步是怎么走的？

```
第n阶
  │
  ├─ 从n-1阶爬1步上来
  │
  └─ 从n-2阶爬2步上来
```

所以：
```
到达n的方法数 = 到达n-1的方法数 + 到达n-2的方法数
dp[n] = dp[n-1] + dp[n-2]
```

## 完整执行图解（n=4）

```
初始化：
dp = [1, 1, 0, 0, 0]  // dp[0]=1(地面), dp[1]=1

第1轮 i=2:
┌─────────────────────────────────────────┐

│                                         │
│  到达第2阶的方法：                       │
│  ├─ 从第1阶迈1步：1种（0→1→2）          │
│  └─ 从第0阶迈2步：1种（0→2）            │
│  共2种                                  │
└─────────────────────────────────────────┘

dp = [1, 1, 2, 0, 0]

第2轮 i=3:
┌─────────────────────────────────────────┐
│  dp[3] = dp[2] + dp[1] = 2 + 1 = 3      │
│                                         │
│  到达第3阶的方法：                       │
│  ├─ 从第2阶迈1步：2种                   │
│  │   ├─ 0→1→2→3                        │
│  │   └─ 0→2→3                          │
│  └─ 从第1阶迈2步：1种（0→1→3）          │
│  共3种                                  │
└─────────────────────────────────────────┘

dp = [1, 1, 2, 3, 0]

第3轮 i=4:
┌─────────────────────────────────────────┐
│  dp[4] = dp[3] + dp[2] = 3 + 2 = 5      │
│                                         │
│  到达第4阶的方法：                       │
│  ├─ 从第3阶迈1步：3种                   │
│  └─ 从第2阶迈2步：2种                   │
│  共5种                                  │
└─────────────────────────────────────────┘

dp = [1, 1, 2, 3, 5]

返回 dp[4] = 5 ✓

**完整 DP 表：**

| i | 0 | 1 | 2 | 3 | 4 |
|---|---|---|---|---|---|
| dp[i] | 1 | 1 | 2 | 3 | 5 |
```

## 边界 Case

**Case 1: n = 0**
```
返回 1（地面，不动也是一种方法）
```

**Case 2: n = 1**
```
返回 1（只有一种方法：迈1步）
```

**Case 3: n = 2**
```
dp[2] = dp[1] + dp[0] = 1 + 1 = 2
方法：1+1, 2
返回 2 ✓
```

## 代码实现

### 基础DP

```java
/**
 * 计算爬楼梯的方法数
 * @param n 楼梯阶数
 * @return 到达第n阶的方法数
 */
public int climbStairs(int n) {
    // 边界情况：0阶或1阶都只有1种方法
    if (n <= 1) return 1;

    // 创建dp数组，dp[i]表示到达第i阶的方法数
    int[] dp = new int[n + 1];

    // 初始条件：第0阶（地面）有1种方法——不动
    dp[0] = 1;

    // 初始条件：第1阶有1种方法——从地面迈1步
    dp[1] = 1;

    // 从第2阶开始递推计算
    for (int i = 2; i <= n; i++) {
        // 状态转移：到达第i阶的方法 = 从i-1迈1步 + 从i-2迈2步
        dp[i] = dp[i-1] + dp[i-2];
    }

    // 返回到达第n阶的方法数
    return dp[n];
}
```

### 空间优化版

```java
/**
 * 空间优化版本：只用两个变量代替整个数组
 * 关键观察：计算dp[i]只需要dp[i-1]和dp[i-2]
 */
public int climbStairsOptimized(int n) {
    // 边界情况
    if (n <= 1) return 1;

    // prev2代表dp[i-2]，初始为dp[0]
    int prev2 = 1;

    // prev1代表dp[i-1]，初始为dp[1]
    int prev1 = 1;

    // 从第2阶开始迭代
    for (int i = 2; i <= n; i++) {
        // 计算当前阶的方法数
        int curr = prev1 + prev2;

        // 滚动更新：prev2变成prev1（前移一位）
        prev2 = prev1;

        // 滚动更新：prev1变成curr（前移一位）
        prev1 = curr;
    }

    // prev1最终保存的是dp[n]
    return prev1;
}
```

## 易错点

### 初始条件容易搞混

| 问题 | dp[0] | dp[1] | 数列 |
|------|-------|-------|------|
| 斐波那契 | 0 | 1 | 0, 1, 1, 2, 3... |
| 爬楼梯 | 1 | 1 | 1, 1, 2, 3, 5... |

**为什么爬楼梯 dp[0]=1？**

`dp[0]` 表示"地面"，有一种方法：不动。

这是为了 `dp[2] = dp[1] + dp[0] = 1 + 1 = 2` 成立。

---

## DP四步法

| 步骤 | 内容 | 本题 |
|------|------|------|
| 1. 定义状态 | dp[i] 表示什么 | dp[i] = 到达第 i 阶的方法数 |
| 2. 状态转移 | dp[i] 怎么算 | dp[i] = dp[i-1] + dp[i-2] |
| 3. 初始条件 | dp[0], dp[1] | dp[0]=1, dp[1]=1 |
| 4. 遍历顺序 | 从哪到哪 | 从 2 到 n |

---

## 举一反三

**如果每次可以爬 1、2 或 3 步呢？**

最后一步有3种可能：
- 从 n-1 爬1步
- 从 n-2 爬2步  
- 从 n-3 爬3步

所以：`dp[n] = dp[n-1] + dp[n-2] + dp[n-3]`
