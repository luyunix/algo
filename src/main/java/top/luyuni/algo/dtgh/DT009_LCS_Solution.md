# DT009 - 最长公共子序列 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：找两个字符串的最长公共子序列（不要求连续）
示例：text1="abcde", text2="ace"
答案：3（LCS是"ace"）

暴力法：枚举text1的所有子序列，检查是否是text2的子序列
- 子序列数量：O(2^m)
- 需要DP
```

### 第二步：分析问题的子结构

```
两个字符串的问题，通常需要二维状态

尝试：dp[i][j] = text1前i个和text2前j个的LCS长度

为什么这样定义？
- 两个串都是独立变化的，需要两个维度
- 子问题明确：两个前缀的LCS
```

### 第三步：从最后字符分类讨论

```
看text1[i-1]和text2[j-1]：

情况1：字符相同
- 这个字符可以加入LCS
- dp[i][j] = dp[i-1][j-1] + 1

情况2：字符不同
- 至少有一个字符不在LCS中
- 取两种选择的最大值：
  - 不用text1[i-1]：dp[i-1][j]
  - 不用text2[j-1]：dp[i][j-1]
- dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

### 第四步：确定初始条件

```
dp[0][j] = 0（text1为空，LCS长度为0）
dp[i][0] = 0（text2为空，LCS长度为0）

答案：dp[m][n]
```

## 核心模型：双串DP

两个字符串的问题，用二维DP。

---

## 递推公式推导

**状态定义**：`dp[i][j] = text1前i个和text2前j个的最长公共子序列长度`

**看最后一个字符**：

```
情况1：text1[i-1] == text2[j-1]
       最后一个字符相同，可以加入LCS
       dp[i][j] = dp[i-1][j-1] + 1

情况2：text1[i-1] != text2[j-1]
       最后一个字符不同，至少有一个不在LCS中
       取两种选择的最大值：
       - 不用text1的最后一个：dp[i-1][j]
       - 不用text2的最后一个：dp[i][j-1]
       dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

## 代码实现

```java
/**
 * 计算两个字符串的最长公共子序列长度
 * @param text1 第一个字符串
 * @param text2 第二个字符串
 * @return 最长公共子序列的长度
 */
public int longestCommonSubsequence(String text1, String text2) {
    // 获取两个字符串的长度
    int m = text1.length();
    int n = text2.length();

    // dp[i][j] = text1前i个字符和text2前j个字符的LCS长度
    // 注意：dp的维度是(m+1) x (n+1)，因为要考虑空字符串的情况
    int[][] dp = new int[m + 1][n + 1];

    // 遍历text1的每个字符（从1开始，对应text1[0]）
    for (int i = 1; i <= m; i++) {
        // 遍历text2的每个字符（从1开始，对应text2[0]）
        for (int j = 1; j <= n; j++) {
            // 获取当前比较的字符
            char c1 = text1.charAt(i - 1);  // text1的第i个字符
            char c2 = text2.charAt(j - 1);  // text2的第j个字符

            if (c1 == c2) {
                // 情况1：两个字符相同
                // 这个字符可以加入LCS，长度+1
                // 继承去掉这两个字符后的LCS长度
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                // 情况2：两个字符不同
                // 至少有一个字符不在LCS中，取两种情况的最大值：
                // 1. 不用text1的当前字符：dp[i-1][j]
                // 2. 不用text2的当前字符：dp[i][j-1]
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }

    // 返回答案：整个text1和整个text2的LCS长度
    return dp[m][n];
}
```

## 边界 Case

**Case 1: 一个字符串为空**
```
text1 = "", text2 = "abc"
没有公共子序列
返回 0 ✓
```

**Case 2: 没有公共字符**
```
text1 = "abc", text2 = "def"
没有相同字符
返回 0 ✓
```

**Case 3: 完全相同**
```
text1 = "abc", text2 = "abc"
整个字符串都是LCS
返回 3 ✓
```

**Case 4: 一个字符**
```
text1 = "a", text2 = "a"
返回 1 ✓

text1 = "a", text2 = "b"
返回 0 ✓
```

**Case 5: 重复字符**
```
text1 = "aaa", text2 = "aa"
LCS是"aa"
返回 2 ✓
```

## DP四步法

| 步骤 | 内容 | 本题 |
|------|------|------|
| 1. 定义状态 | dp[i][j] 表示什么 | dp[i][j] = text1 前 i 个和 text2 前 j 个的 LCS 长度 |
| 2. 状态转移 | dp[i][j] 怎么算 | 字符相同：dp[i-1][j-1]+1<br/>字符不同：max(dp[i-1][j], dp[i][j-1]) |
| 3. 初始条件 | dp[0][j], dp[i][0] | 都是 0 |
| 4. 遍历顺序 | 从哪到哪 | i: 1→m, j: 1→n |

---

## 和编辑距离的对比

| | LCS | 编辑距离 |
|--|-----|----------|
| 字符相同 | dp[i-1][j-1] + 1 | dp[i-1][j-1] |
| 字符不同 | max(dp[i-1][j], dp[i][j-1]) | min(替换,删除,插入) + 1 |
| 目标 | 最大化 | 最小化 |

## 记忆口诀

```
双串DP看末尾，相同就加1
不同取max，不用这个或那个
LCS要最长，编辑距离要最短
```
