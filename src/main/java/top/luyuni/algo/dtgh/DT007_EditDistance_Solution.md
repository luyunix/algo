# DT007 - 编辑距离 题解

## 状态定义是怎么想出来的

### 第一步：明确问题

```
问题：word1转成word2的最少操作数（插入、删除、替换）
示例：word1="horse", word2="ros"
答案：3（horse→rorse→rose→ros）

暴力法：枚举所有可能的操作序列
- 操作序列爆炸，需要DP
```

### 第二步：分析问题的子结构

```
关键观察：两个字符串的问题，通常需要二维状态

为什么一维不够？
- word1的前i个和word2的前j个
- i和j是独立变化的，需要两个维度

尝试：dp[i][j] = word1前i个转成word2前j个的最少操作数
```

### 第三步：从最后字符倒推

```
看两个字符串的最后一个字符：
word1[i-1] 和 word2[j-1]

情况1：字符相同
- 不需要操作，直接看前面
- dp[i][j] = dp[i-1][j-1]

情况2：字符不同
- 替换：把word1[i-1]换成word2[j-1]，然后看dp[i-1][j-1]
- 删除：删掉word1[i-1]，然后看dp[i-1][j]
- 插入：在word1末尾插入word2[j-1]，然后看dp[i][j-1]
- dp[i][j] = min(替换, 删除, 插入) + 1
```

### 第四步：确定初始条件

```
dp[i][0] = i（word1前i个转成空串，需要删除i次）
dp[0][j] = j（空串转成word2前j个，需要插入j次）
```

## 核心模型：双串DP

两个字符串的问题，用二维DP：`dp[i][j]` 表示前i个和前j个的关系。

---

## 递推公式推导

**看最后一个字符**：

```
word1 = "hor" (i=3)
word2 = "ro"  (j=2)

最后一个字符：word1[2]='r', word2[1]='o'
```

**情况1：最后一个字符相同**
```
word1 = "ho", word2 = "ro"
最后一个都是 'o'

不需要操作，直接看前面：
dp[2][2] = dp[1][1]
```

**情况2：最后一个字符不同**
```
有三种选择：

1. 替换：把 'r' 换成 'o'
   "hor" -> "hoo"，然后看 "ho" -> "h"
   代价：dp[i-1][j-1] + 1

2. 删除：删掉 'r'
   "hor" -> "ho"，然后看 "ho" -> "ro"
   代价：dp[i-1][j] + 1

3. 插入：插入 'o'
   "hor" -> "horo"，然后看 "hor" -> "r"
   代价：dp[i][j-1] + 1

取最小值！
```

## 代码实现

```java
/**
 * 计算将一个字符串转换成另一个字符串的最少操作数
 * 支持的操作：插入、删除、替换
 * @param word1 源字符串
 * @param word2 目标字符串
 * @return 最少操作数
 */
public int minDistance(String word1, String word2) {
    // 获取两个字符串的长度
    int m = word1.length();
    int n = word2.length();

    // dp[i][j] = word1的前i个字符转成word2的前j个字符的最少操作数
    // 注意：dp的索引比字符串索引大1，dp[i][j]对应word1[i-1]和word2[j-1]
    int[][] dp = new int[m + 1][n + 1];

    // ========== 初始条件 ==========

    // word1的前i个字符转成空字符串：需要删除i个字符
    for (int i = 0; i <= m; i++) {
        dp[i][0] = i;
    }

    // 空字符串转成word2的前j个字符：需要插入j个字符
    for (int j = 0; j <= n; j++) {
        dp[0][j] = j;
    }

    // ========== 状态转移 ==========

    // 填充dp表，i从1到m，j从1到n
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            // 获取当前比较的字符
            char c1 = word1.charAt(i - 1);  // word1的第i个字符
            char c2 = word2.charAt(j - 1);  // word2的第j个字符

            if (c1 == c2) {
                // 情况1：两个字符相同，不需要操作
                // 直接继承之前的结果
                dp[i][j] = dp[i - 1][j - 1];
            } else {
                // 情况2：两个字符不同，取三种操作的最小值+1

                // 替换操作：把c1替换成c2，然后继续比较前面的部分
                int replace = dp[i - 1][j - 1];

                // 删除操作：删除c1，然后比较word1的前i-1和word2的前j
                int delete = dp[i - 1][j];

                // 插入操作：在word1的i位置插入c2，然后比较word1的前i和word2的前j-1
                int insert = dp[i][j - 1];

                // 取三种操作的最小值，然后+1（当前这次操作）
                dp[i][j] = Math.min(replace, Math.min(delete, insert)) + 1;
            }
        }
    }

    // 返回答案：整个word1转成整个word2的最少操作数
    return dp[m][n];
}
```

## 边界 Case

**Case 1: 一个字符串为空**
```
word1 = "", word2 = "abc"
需要插入3个字符
返回 3 ✓

word1 = "abc", word2 = ""
需要删除3个字符
返回 3 ✓
```

**Case 2: 两个字符串相同**
```
word1 = "abc", word2 = "abc"
不需要任何操作
返回 0 ✓
```

**Case 3: 完全不同**
```
word1 = "abc", word2 = "def"
需要替换3次
返回 3 ✓
```

**Case 4: 单字符**
```
word1 = "a", word2 = "b"
替换1次
返回 1 ✓

word1 = "a", word2 = "a"
相同，返回 0 ✓
```

## DP四步法

| 步骤 | 内容 | 本题 |
|------|------|------|
| 1. 定义状态 | dp[i][j] 表示什么 | dp[i][j] = word1 前 i 个转 word2 前 j 个的最少操作数 |
| 2. 状态转移 | dp[i][j] 怎么算 | 字符相同：dp[i-1][j-1]<br/>字符不同：min(替换，删除，插入)+1 |
| 3. 初始条件 | dp[i][0], dp[0][j] | dp[i][0]=i, dp[0][j]=j |
| 4. 遍历顺序 | 从哪到哪 | i: 1→m, j: 1→n |

---

## 易错点详解

### 1. 数组大小是 (m+1) x (n+1)

```java
// 错误
int[][] dp = new int[m][n];

// 正确
int[][] dp = new int[m + 1][n + 1];  // 要包含空字符串的情况
```

### 2. 索引差1

```java
// dp[i][j] 对应 word1[i-1] 和 word2[j-1]
word1.charAt(i - 1) == word2.charAt(j - 1)
```

### 3. 三种操作的理解

```
dp[i-1][j-1]：替换
  - 把word1[i-1]换成word2[j-1]
  - 然后看word1[0..i-2]和word2[0..j-2]

dp[i-1][j]：删除
  - 删掉word1[i-1]
  - 然后看word1[0..i-2]和word2[0..j-1]

dp[i][j-1]：插入
  - 在word1末尾插入word2[j-1]
  - 然后看word1[0..i-1]和word2[0..j-2]
```
